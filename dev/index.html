polar-bits#

Bluesky-based data acquisition for the POLAR beamline (4ID) at the Advanced Photon Source.

polar-bits is built on the BITS (Bluesky Instrument Toolkit Structure) framework and provides a shared device library, scan plans, and data output callbacks for four hutches: 4IDB, 4IDG, 4IDH, and Raman.
