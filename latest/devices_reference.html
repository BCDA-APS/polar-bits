Device Reference#

All devices are defined in src/id4_common/configs/devices.yml and loaded at session startup based on their labels. Use find_loadable_devices() to query available devices interactively.

find_loadable_devices()                    # all devices
find_loadable_devices(label="4idg")        # filter by hutch
find_loadable_devices(label="detector")    # filter by function
find_loadable_devices(name="kb")           # substring search by name
